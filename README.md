## Projets
 
### 🔐 [Cryptobox](https://github.com/fafifou03/cryptobox) — *à venir*
Outil de transfert de fichiers chiffrés. Chiffrement/déchiffrement, génération d'URL unique, gestion des accès non autorisés, journalisation des échanges.
`Python` `JavaScript`

 ### 🌐 Extracteur d'IoCs — MISP — *à venir*
Extraction automatisée d'indicateurs de compromission depuis une instance MISP locale via PyMISP. Projet réalisé pour apprendre l'interaction avec des APIs REST et m'initier au threat intelligence.
`Python` `PyMISP`

### ⚡ [Energy Optimizer Campus](https://github.com/fafifou03/energy-optimizer-campus)
Simulateur pédagogique d'optimisation énergétique. Frontend React/TypeScript, backend Python FastAPI.
`Python` `TypeScript` `React`
 
### 📊 [Analyse de réseaux — Théorie des graphes](https://github.com/fafifou03/Projet-Theorie-des-graphes-)
Modélisation d'une commune en graphe pondéré. Implémentation de BFS/DFS, Kosaraju, connectivité. Pilotage d'une équipe de 4.
`Java`
 
### ♟️ [Jeu du Quoridor](https://github.com/fafifou03/Projet-Jeu-du-Quoridor-en-C)
Adaptation console du jeu Quoridor avec gestion des murs, des déplacements et du plateau. Un de mes premiers projets en informatiqueC.
`C`
